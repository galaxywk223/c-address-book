# 数据文件格式说明

## 文件格式
数据使用 CSV（逗号分隔值）格式存储，文件编码为 UTF-8（带 BOM）。

## 字段说明

| 序号 | 字段名 | 英文名 | 类型 | 必填 | 说明 |
|------|--------|--------|------|------|------|
| 1 | 编号 | id | 整数 | 是 | 唯一标识，自动递增 |
| 2 | 姓名 | name | 字符串 | 是 | 最大 49 字符 |
| 3 | 单位 | company | 字符串 | 否 | 最大 99 字符 |
| 4 | 固定电话 | phone | 字符串 | 否 | 支持数字、横线、括号 |
| 5 | 手机号 | mobile | 字符串 | 否 | 11 位数字 |
| 6 | 分类 | category | 整数 | 是 | 0=家人 1=朋友 2=同事 3=商业 4=其他 |
| 7 | Email | email | 字符串 | 否 | 基本邮箱格式验证 |
| 8 | QQ | qq | 字符串 | 否 | 5-12 位数字 |
| 9 | 地址 | address | 字符串 | 否 | 最大 199 字符 |
| 10 | 备注 | note | 字符串 | 否 | 最大 199 字符 |
| 11 | 创建时间 | create_time | 时间戳 | 是 | Unix 时间戳 |
| 12 | 修改时间 | modify_time | 时间戳 | 是 | Unix 时间戳 |

## CSV 特殊处理

- 包含逗号、双引号或换行符的字段会用双引号包裹
- 字段内的双引号用两个双引号转义（`""`）
- 文件开头包含 UTF-8 BOM（`\xEF\xBB\xBF`）以兼容 Excel 打开中文

## 示例行

```
1,张伟,北京科技有限公司,010-88886666,13800138001,2,zhangwei@example.com,123456789,北京市海淀区中关村大街1号,同事,1717000000,1717000000
```

## 文件路径

- 主数据文件：`data/contacts.csv`
- 回收站文件：`data/trash.csv`
- 备份目录：`data/backup/`
- 导出文件：`data/export.csv`（默认）
